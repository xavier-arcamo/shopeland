# Shopeland

Shopeland is a browser RTS prototype with a medieval sheep-vs-wolf theme.

## Highlights
- Isometric 3D battlefield built with Three.js.
- External map file (`map.json`) loaded at runtime.
- Terrain features: rolling elevation/slopes, rivers, lakes, mountains, forests, settlements, towers, and bridges.
- Total-war-like division control: drag selection + bottom unit cards for multi-select control groups.
- RTS commands: right-click formation move, attack-move, mass attack, regroup, recruitment economy, wave defense.

## Run
```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.
